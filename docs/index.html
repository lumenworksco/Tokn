Built to be trusted

Open source, zero dependencies, auditable in an afternoon.

📊

Two limits at a glance

Shows your 5-hour session and 7-day weekly usage side by side, with exact reset times — no need to open claude.ai.

🔒

Keychain-only storage

Your sk-ant-... session key is stored exclusively in the macOS Keychain. It is never written to disk or sent anywhere other than claude.ai.

🎨

Colour-coded status

The menu bar dot and usage bars shift from green to orange (≥50%) to red (≥80%) so you can tell at a glance how much runway you have left.

🔍

Fully auditable

~20 Swift files, zero third-party dependencies. The entire network layer fits in one file — read exactly what requests are made and what headers are sent.

⚡

Auto-refresh

Polls the API on a configurable interval (1–10 minutes). Starts automatically at login and restarts the loop whenever you change the interval.

🔄

Auto-update

Checks for new versions on every launch. When an update is available, a one-click banner appears — Tokn downloads, installs, and relaunches itself automatically.

🚫

Zero telemetry

No analytics, no crash reporters. The only outbound traffic is API calls to claude.ai using your own session key, plus a single GitHub releases check on launch.

📈

Usage trend & ETA

Tap the sparkline on any card to expand a full history chart with warning and critical threshold markers. After a few minutes of use, the card footer shows a live ETA — "~2h at this pace" — when you're on track to hit the limit before the window resets.

🔔

Smart notifications

Get alerted before you hit the cap. The first threshold is yours to set: 50%, 60%, 70%, 80%, or 90%. A second alert fires at 100%. Tracking resets automatically when usage drops, so each new spike notifies again.
